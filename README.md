# LLM Context Optimization using Memory Compression

## Description
A LangChain-based system that optimizes LLM context by compressing conversation memory, reducing token usage, and improving response efficiency.

## Tech Stack
Python, LangChain, OpenAI API, Prompt Engineering, Memory Management

## Features
- Dynamic memory compression
- Context optimization
- Reduced token usage
- Efficient long conversations

## Project Structure
- llm.py → LLM setup
- compression.py → core logic
- main.py → execution
- prompts.py → prompt templates
